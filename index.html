Co je Cloudflare?

Stručné vysvětlení v pár větách

Cloudflare je americká technologická společnost, která provozuje jednu z největších globálních sítí pro doručování obsahu a ochranu webů. Funguje jako tzv. „reverzní proxy" – stojí mezi návštěvníky internetu a servery webových stránek, přičemž filtruje provoz, chrání před útoky a urychluje načítání. Díky síti více než 300 datových center rozmístěných po celém světě dokáže obsloužit stovky miliard internetových požadavků denně. Základní verze služeb je přitom dostupná zdarma pro každého.

K čemu Cloudflare slouží?

Čtyři hlavní oblasti, ve kterých Cloudflare pomáhá

🛡️

Bezpečnost

Chrání weby před kybernetickými útoky, škodlivými roboty a neoprávněným přístupem – automaticky a bez složitého nastavení.

⚡

Výkon

Díky globální CDN síti se obsah webu načítá z nejbližšího serveru na světě, takže stránky se uživatelům zobrazují výrazně rychleji.

🌐

Dostupnost

Pokud váš hosting selže, Cloudflare může stále obsluhovat návštěvníky z mezipaměti – web tak zůstane online i při výpadku serveru.

🔒

Soukromí a HTTPS

Automaticky vydává SSL/TLS certifikáty, takže komunikace mezi uživatelem a webem je vždy zašifrovaná a bezpečná.

Vybrané služby

Co Cloudflare konkrétně nabízí

Služba 01

CDN – Content Delivery Network

Cloudflare ukládá kopie vašeho webu do datových center po celém světě, takže návštěvníci stahují obsah z geograficky nejbližšího serveru a stránky se načítají mnohem rychleji.

Výkon
Služba 02

DDoS ochrana

Cloudflare automaticky detekuje a blokuje distribuované útoky (DDoS), které se snaží web zahltit obrovským množstvím požadavků – legitimní návštěvníci tak nejsou nijak ovlivněni.

Bezpečnost
Služba 03

DNS resolver

Cloudflare provozuje veřejný DNS resolver na adrese 1.1.1.1, který patří k nejrychlejším a nejsoukromějším DNS službám na světě – neukládá záznamy o vašich dotazech.

Soukromí
Služba 04

SSL / TLS certifikáty

Cloudflare automaticky zajistí bezplatný SSL certifikát pro váš web, takže přechod na šifrovaný protokol HTTPS je otázkou jediného kliknutí bez nutnosti technických znalostí.

Bezpečnost
Služba 05

Web Application Firewall (WAF)

Firewall webových aplikací filtruje škodlivé HTTP požadavky a chrání web před běžnými zranitelnostmi, jako jsou SQL injection nebo cross-site scripting (XSS).

Bezpečnost
Služba 06

Cloudflare Pages

Cloudflare Pages je platforma pro hostování statických webových stránek přímo z repozitáře na GitHubu – nasazení je automatické po každém commitu a hosting je zdarma.
